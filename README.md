# Advanced Image Gallery App

A responsive Image Gallery web application built using HTML, CSS, and JavaScript. This project was created for frontend internship practice and portfolio demonstration.

## Features

* Responsive grid layout
* Light and dark mode
* Image search by caption
* Category filtering (Nature, City, Animals)
* Hover effects with zoom and caption overlay
* Full-screen lightbox preview
* Next and previous navigation
* Image upload from local device

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
